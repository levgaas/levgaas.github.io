jPhotoGrid.js

By Vladimir Barsuchenko

What is it?

jPhotoGrid.js is a jQuery plugin that helps you create a perfect grid with no gaps and no ragged edges. It Works by mapping elements in a grid, finding where gaps are and filling them in as a stone mason would do.

jPhotoGrid.js is not Masonry, Isotope, Packery, Gridilicious or any of those other grid plugins. It is for creating perfect grids.

Examples

Below is a basic example of jPhotoGrid.js at work. 

$(".pictures").jPhotoGrid({
        margin: 1,
        isFirstRowBig: true,
        isCentred: true,
        isSmallImageStretched: false,
        onComplete: function () {},
        onProgress: function (instance, image) {}
    });
