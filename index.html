jPhotoGrid.js

By Vladimir Barsuchenko

What is it?

jPhotoGrid.js is a jQuery plugin that helps you create a perfect image grid with no gaps and no ragged edges. Content is wrapped by block with fixed height and width. So it doesn't matter how many images you have. All of them will be aligned by height inside block.

jPhotoGrid.js plugin doesn't guarantee that images will be stretched on wrapper height. Because image height is calculated from width * ratio. But it does guarantee stretching by width.

jPhotoGrid.js is not Masonry, Mason, Isotope, Packery, Gridilicious or any of those other grid plugins. It is for creating perfect grids with fixed height.

Examples

Below are basic examples of jPhotoGrid.js at work with different amount of images.

You can click on image to delete it. 

$(".pictures").jPhotoGrid({
        margin: 1,
        isFirstRowBig: true,
        isCentred: true,
        isSmallImageStretched: false,
        onComplete: function () {},
        onProgress: function (instance, image) {}
    });

Options

Number

0

First row - largest

false

isCentred

Center content if overall height less than wrapper height

true

isSmallImageStretched

Stretch image if it has dimensions smaller than wrapper's one

false

Min row relation = Height / Width

0.4

{}
